个人博客(基础)

发表于 2018-07-17 | 分类于 hexo

主要技术是：github+hexo+next主题。

主要步骤

1、安装和配置Git环境

下载：Git
配置成功，利用代码验证 

$ git --version
2、安装和配置Node.js环境

下载：Node.js
配置成功，利用代码验证 

$ node -v       
$ npm -v
3、新建Github项目

官网：Github
项目的名称必须是：username.github.io，需要勾选Initialize this repository with a README
在新建的项目的settings中，找到Github Pages,会看到网址，通过该网址可以访问个人博客 (url:username.github.io/.)

4、安装Hexo

(1)在合适的位置新建一个blog文件夹，通过cmd或者Git bash进入blog文件夹
安装Hexo 

$ npm install hexo -g

(2)检验是否安装成功 

$ hexo -v

(3)初始化blog文件夹 

$ hexo init

(4)会看到后面出现“Start blogging with Hexo!”
安装所需要的组件 

$ npm install

(5)体验hexo 

$ hexo g

(6)开始服务器，不要按Ctrl+C否则无法正常访问 

$ hexo s

在浏览器中输入 url:localhost:4000,可以访问该网址
至此，Hexo部分的工作已经完成。

5、Hexo与Github连接

(1)第一次使用Git需要设置username和email 

$ git config --global user.name "username"    
$ git config --global user.email"xxx@email.com"

(2)配置公钥 

$ ssh-keygen -t rsa -C "xxx@qq.com"

生成密钥，得到了两个文件：id_rsa和id_rsa.pub（默认存储路径是：C:\Users\Administrator.ssh）。
(3)添加密钥到ssh-agent 

$ ssh-add ~/.ssh/id_rsa

(4)将ssh添加到Github
在settings新建一个new ssh key，将id_rsa.pub文件里的内容复制上去
(5)测试添加ssh是否成功 

$ ssh -T git@github.com

出现用户名，说明成功了。
(6)在blog文件夹中找到_config.yml文件，修改repo值 

deploy:    
  type: git    
  repository: git@github.com:username/username.github.io.git    
  branch: master

这里注意冒号后边一定空一格。
(7)安装扩展 

$ npm install hexo-deployer-git --save

(8)生成及部署 

$ hexo d     
$ hexo g

部署成功后可以访问 url:username.github.io
hexo+github的个人博客搭建完成。
